# 🧠 Elden Ring Chatbot

A React-based frontend for an AI-powered chatbot trained on *Elden Ring* knowledge — including lore, weapons, demigods, and locations.

---

## 🚀 Overview

This project provides the user interface for an AI chatbot that answers questions about the world of **Elden Ring**.

The backend (handled separately) will connect a language model with a curated knowledge base (PDF-based) to deliver accurate, context-aware responses.

---

## 🧩 Features (Planned)

* 💬 Real-time chat interface
* 🤖 AI-generated answers based on Elden Ring data
* 📚 Knowledge grounded in custom PDF sources
* 🎯 Structured message system (user / bot)
* ⚡ Fast and lightweight frontend (Vite + React)

---

## 🛠️ Tech Stack

* **React** (with TypeScript)
* **Vite** (build tool)
* **Git + GitHub** (version control)

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/haslingerfabian/Elden-Ring-Chatbot.git
cd elden-ring-chatbot
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## 🧠 Concept

The chatbot will be powered by a language model enhanced with external knowledge:

1. User sends a question
2. Backend retrieves relevant information from PDF
3. AI generates a structured response
4. Frontend displays the conversation
